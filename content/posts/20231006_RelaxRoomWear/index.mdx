# 発売前の衣装を着ませんか。

コロナワクチン接種の翌日はいつも微熱が出るので、仕事を休むことにしている。\
\
木曜夕方にワクチンを接種し、翌金曜日、平日に休んだ日の夜のこと。衣装製作をされている あいかわさん より、「明日、新衣装を発売するんですけど良ければ着ませんか？？」とのDMが届きました。\
しかも、「良かったらおはつい等で使ってもらえたら嬉しいなと思って…！！」とのこと。\
\
これがもう自分にとってはめちゃくちゃうれしかった。\
\
そもそも自分のおはツイは良いの撮れたら出すか～～仲良しのフォロワーさんにいいねもらえたらいいな。\
ぐらいの気持ちだし、よく投稿をお休みして飛び飛びになったりしているので、認識してもらえているとはあまり思っていなかった所に、このようなお話を頂いて、ありがたい限り………\
\
しかも、ちょうどお休みを使って、ルナトくんアバターの衣装を着やすくするための git リポジトリを再作成していたタイミングだったので、即着ます。と返事をしたのでした。\
\
そこからは急いで改変をし、フレンドに見せびらかし、1週間はおはツイできるぐらいに写真をめっちゃ撮ったら朝7時……という感じでした。\
（急いで改変したので、インポートしたアセットによってUTS2が変化していることに気づかず、髪留めが光っていない。などのミスがありますが…まあいいかな………）\
\
という事で、一週間分のおはツイと使わなかった写真をここにまとめます。\
もし、このような機会をまた頂けるのであれば、次も頑張りたいです。

# いただいた衣装
Booth で買えます。男の子でも着られるかわいい部屋着は少ないので皆さんも是非…！！
